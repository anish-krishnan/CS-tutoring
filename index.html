Categories

We don't have a structured curriculum
We currate our curriculum to your needs and goals.
If you're goal is to train an ML model to detect cats in pictures, we're here to teach you the fundamentals of how to accomplish that.

  • Programming in an imperative language (i.e. Python, C, and Java)
  • Programming in a functional language (i.e. SML) 
  • Theoretical CS/CS theory (Turing machines, P=NP, graph theory, time complexity, etc.) 
  • Algorithms (randomized/approximation algorithms, big-oh, amortized analysis, Dynamic Programming, etc.) 
  • Artificial Intelligence (machine learning, game theory, linear programming, etc.)
  • Data structures (hashing, linked lists, trees, etc.)
  • Systems (caching, memory management, networks, etc.) 
  • Web development (make your own website!)
